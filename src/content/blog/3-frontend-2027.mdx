## Astro, Next.js y el Fin del SPA

Cada vez más empresas están dándose cuenta que el rendimiento en móviles es igual o más importante que la estética.

Es aquí donde el Single Page Application puro (SPA, como React sin meta-framework) cede paso al **Server Side Rendering** (SSR) y Static Site Generators (SSG), como Astro, el campeón absoluto para sitios web orientados al contenido y velocidad pura, enviando 0 Javascript a clientes innecesariamente.

## Micro-Interacciones de CSS

Ya no dependemos de pesadas librerías de JS para lograr animaciones complejas. Usando CSS nativo (`View Transitions API`, `scroll-timeline`) le damos al usuario una experiencia web casi equiparable a una aplicación nativa instalada en su dispositivo móvil.
